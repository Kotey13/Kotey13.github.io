Twitter Bootstrap
Cвободный набор инструментов для создания сайтов и веб‐приложений
Включает в себя HTML и CSS шаблоны оформления для типографики, веб‐форм, кнопок, меток,блоков навигации и прочих компонентов веб‐интерфейсов,включая JavaScript расширения. Bootstrap использует самые современные наработки в области CSS и HTML, поэтому необходимо быть внимательным при поддержке старых браузеров.
